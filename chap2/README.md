1. ncv 이용 개발 제품 및 기능
 - 운전자 졸음감지 시스템: 전방 카메라를 통한 운자자의 눈 깜박임, 얼굴 방향을 분석하여 
			   경고음 알림
 - 얼굴 인식 및 인증 시스템: 얼굴을 탐지 및 신원 식별 ex) 스마트폰 잠금해제

2. 소프트웨어 분야의 용어인 라이브러리의 의미
- 많이 사용되는 기능을 함수 및 클래스로 정의해 놓은 것 
 -> 헤더파일(함수,클래스선언)과 목적파일(함수정의, 객체선언)로 구성

3. 오픈소스의 의미
- 오픈소스로 개발된 컴퓨터 비전 라이브러리

4. Windows 운영체제의 환경변수 중 Path 변수의 기능을 설명하라
- 운영체제가 명령 프롬프트나 실행팡을 찾을 때 참조되는 디렉토리 경로 

5.프로그램 개발과정인 편집, 컴파일, 링크, 실행 단계를 각각 설명하고 링크단계에서의
  라이브러리의 역할에 대하여 설명하라
- 편집: 프로그래머가 프로그램언어를 사용하여 소스코드를 작성하는 단계
- 컴파일: 소스코드를 기계어로 번역하는 과정
- 링크: 컴파일되어 생성된 하나 이사으이 목적파일과 필요한 라이브러리 파일들을 묶어
        하나의 최종 실행 파일을 만드는 과정
- 실행단계: 링크를 통해 생성된 최종 실행파일을 실행하는 단계

링크단계에서의 라이브러리 역할: dll을 로드하고 호출하는 기능

6. 정적 라이브러리(SLL)와 동적 라이브러리(DLL)에 대하여 조사하고, 각 라이브러리가
   실행파일에 링크되는 시점을 자세히 설명하라
- 동적링크 라이브러리: dll 실행단계에서 실행파일과 연결되어 실행됨
- 정적링크 라이브러리: lib 링크단계에서 사용자 목적파일과 결합하여 실행파일 생성

7. 동적 라이브러리 (DLL)의 경로는 운영체제의 환경변수 PATH에 등록되어야 하는 이유설명
- 응용 프로그램을 실행하는 위해서는 OpenCV DLL 파일이 필요하기 때문
